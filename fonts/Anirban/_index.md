# Anirban

Anirban is mainly a display font. It has four styles. These are regular, italic, bold and bold italic. Which will be very much useful for the user and designer. The visual characteristic of this typeface specially indicates something strong. You can also use it as a paragraph font.

## Font Information

- **Font Name:** Anirban
- **Styles:** 4 Styles
- **Category:** Paragraph, Display
- **Language Support:** Bangla
- **Encoding:** UNICODE
- **Designer:** Imrul Kayes Sourav
- **Version:** 1.0
- **License:** Open Source
