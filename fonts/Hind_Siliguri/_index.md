# Hind Siliguri

Hind Siliguri is a free Bengali typeface developed by the Indian Type Foundry (ITF). It is a part of the Hind font family and is designed for use in both digital and print media, offering a clean and modern sans-serif look for the Bengali script.

## Font Information

- **Font Name:** Hind Siliguri
- **Designer:** Indian Type Foundry
- **Version:** 1.0
- **License:** SIL Open Font License (OFL) 1.1
